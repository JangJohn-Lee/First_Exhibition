# 할머니의 첫 개인전: 윤종란 개인전

이 프로젝트는 87세 윤종란 할머니의 첫 개인전을 위해 제작된 웹사이트입니다. 새벽 2시, 손님이 두고 간 펜으로 달력 뒷면에 고향의 봄을 그리며 시작된 그림들을 모아 온라인 전시회로 꾸몄습니다.

## 🎨 전시 소개

**"새벽2시, 손님이 두고 간 펜으로 그린 고향의 봄"**

이번 전시는 제대로 된 그림 도구 하나 없이, 버려진 볼펜과 달력 뒷면을 캔버스 삼아 피어난 할머니의 그림들을 선보입니다. 그리운 고향, 그날 본 꽃과 나무, 정겨운 동물과 집 등 소박하지만 따뜻한 시선이 담긴 작품들을 만나보실 수 있습니다.

> "잠이 안 와가지고... 우리 집 한번 그려볼까? 고향이 그립더라고. 그래서 고향이 그리워서 그린 거라."

## ✨ 주요 기능

- **카카오톡 공유하기**: 마음에 드는 작품이나 전시회 자체를 카카오톡으로 친구들에게 공유할 수 있습니다.
- **전시 정보 공유하기**: 현재 보고 있는 전시회 페이지의 링크를 간편하게 공유할 수 있습니다.
